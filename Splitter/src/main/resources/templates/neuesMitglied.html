Neues Mitglied hinzufügen

Hier kannst du ein neues Mitglied in die Gruppe "Dummy-Gruppe" hinzufügen.

Bitte lass dir dafür von deinem/deiner Freund*in seinen/ihren GitHub Handle geben.

Folgende Regeln gelten für GitHub Namen:

  • Ein valider GitHub Username besteht aus beliebigen Zahlen und Buchstaben.
  • Umlaute (ä/ö/ü) sind nicht erlaubt.
  • Die Groß-/Kleinschreibung spielt keine Rolle.
  • Neben Buchstaben und Zahlen sind auch Bindestriche (-) erlaubt, es dürfen aber keine zwei Bindestriche aufeinander folgen und ein Handle kann nicht mit einem Bindestrich anfangen/enden.
